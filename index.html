Installation Guide

Choose your installation method below. Follow each step carefully.

‚ö†Critical Warning

BACKUP YOUR DATA FIRST! This process will completely wipe your device. Make sure you have:

  • Full device backup (photos, contacts, apps)
  • Your Google/Samsung account ready for restore
  • Important files saved to cloud or PC

1Prerequisites & Downloads

Before starting, ensure you have these tools ready:

  • Unlocked bootloader on your device
  • Samsung Odin (latest version from developer)
  • Samsung USB drivers installed on PC
  • OneAI ROM for your exact device model
  • Quality USB-C cable (avoid cheap cables)

2Prepare Your Device

Set up your device for flashing:

  • Go to Settings ‚Üí Developer Options
  • Enable USB Debugging
  • Enable OEM Unlocking
  • Power off your device completely
  • Boot into Download Mode: Hold Volume Down + Volume Up + Power while connecting USB cable

3Flash with Odin

IMPORTANT: Use CSC file, NOT HOME_CSC for clean installation!

  • Open Odin as Administrator on your PC
  • Connect device (should show blue "Added" message)
  • Load OneAI files in correct slots:
    • AP: OneAI_AP_*.tar.md5
    • BL: OneAI_BL_*.tar.md5
    • CP: OneAI_CP_*.tar.md5
    • CSC: OneAI_CSC_*.tar.md5 (NOT HOME_CSC!)
  • Click Start and wait (10-15 minutes)
  • Device will reboot automatically when done

4Format Data & First Boot

Complete the installation:

  • Device should boot to TWRP recovery automatically
  • Go to Wipe ‚Üí Format Data
  • Type yes to confirm data wipe
  • Select Reboot System
  • First boot takes 10-15 minutes - be patient!
  • Complete Android setup and enjoy OneAI!

‚ö†Coming from Custom ROM

Installing over other custom ROMs requires extra care. Clean installation is highly recommended to avoid conflicts and bootloops.

1Complete Wipe Required

Unlike stock ROM, coming from custom ROM needs full cleanup:

  • Boot into TWRP recovery
  • Go to Wipe ‚Üí Format Data
  • Type yes to confirm complete data format

2Flash OneAI ROM

Now flash the ROM with Odin:

  • Reboot to Download Mode
  • Connect to PC and open Odin
  • Load OneAI ROM files in appropriate slots
  • Use CSC file (NOT HOME_CSC) for clean install
  • Click Start and wait for completion

3Final Steps

Complete the installation:

  • Device should boot to TWRP automatically
  • If you see any errors, Format Data again
  • Reboot System
  • First boot will take 10-15 minutes
  • If stuck on boot screen, boot back to TWRP and format data once more

‚úìEasy Update Process

Updating from older OneAI versions is much simpler! Your data and settings can be preserved in most cases.

1Download Latest ROM

Get the newest OneAI version:

  • Download the latest OneAI ROM for your device
  • Ensure it's the correct model (G991B, G996B, etc.)
  • Have Odin ready on your PC

2Flash Update

Key difference: Use HOME_CSC this time to preserve data!

  • Boot into Download Mode
  • Connect to PC and open Odin
  • Load ROM files in slots
  • Use HOME_CSC file (not regular CSC)
  • This preserves your apps and data
  • Click Start and wait for completion

3Reboot and Enjoy

Simple finish:

  • Device will reboot automatically
  • Your apps and settings should be intact
  • OneAI will show new version in Settings
  • No setup required - you're done!

üîßTroubleshooting

If you experience issues after updating:

  • Bootloop or won't start:
    • Boot into TWRP recovery
    • Go to Wipe ‚Üí Format Data
    • Type yes to confirm
    • Reboot system
  • Apps crashing: Clear cache in Settings ‚Üí Storage
  • Performance issues: Restart device and wait 24h for optimization
