Installation Guide

⚠ Critical Backup your data first!

This process will completely wipe your device. Before flashing, make sure you have:

  • Full device backup (photos, contacts, apps)
  • Google / Samsung account ready for restore
  • Important files saved to cloud or PC
1 Prerequisites & Downloads

Make sure you have:

  • Unlocked bootloader on your device
  • Samsung Odin (latest version)
  • Samsung USB Drivers installed on PC
  • OneAI ROM for your exact device model
  • Quality USB-C cable
2 Prepare your device
  • Go to Settings → Developer Options
  • Enable USB Debugging
  • Enable OEM Unlocking
  • Power off your device completely
  • Boot into Download Mode: hold Vol Down + Vol Up + Power while connecting USB
3 Flash with Odin

Use CSC — NOT HOME_CSC for a clean install!

  • Open Odin as Administrator on PC
  • Connect device — should show blue "Added!" message
  • Load files in correct slots:
Odin slots 
AP:  OneAI_AP_*.tar.md5
BL:  OneAI_BL_*.tar.md5
CP:  OneAI_CP_*.tar.md5
CSC: OneAI_CSC_*.tar.md5
  • Click Start and wait 10–15 minutes
  • Device reboots automatically when done
4 Format Data & First Boot
  • Device boots to TWRP recovery automatically
  • Go to Wipe → Format Data
  • Type yes to confirm
  • Select Reboot System
  • First boot takes 10–15 minutes — be patient!
⚠ From Custom ROM — Extra care needed

Coming from another custom ROM requires a full clean wipe to avoid bootloops and conflicts.

1 Full wipe in TWRP
  • Boot into TWRP recovery
  • Go to Wipe → Format Data
  • Type yes to confirm
2 Flash OneAI with Odin
  • Reboot to Download Mode
  • Connect to PC and open Odin
  • Load ROM files, use CSC (NOT HOME_CSC)
  • Click Start and wait
3 Final Steps
  • Device boots to TWRP automatically
  • If errors appear, run Format Data again
  • Reboot System — first boot: 10–15 minutes
✓ Easy Update — Data Preserved

Updating from an older OneAI version is much simpler. Your data and settings can be preserved in most cases.

1 Download latest ROM
  • Download the latest OneAI for your exact model
  • Have Odin ready on PC
2 Flash with HOME_CSC

Key difference: Use HOME_CSC this time to preserve data!

Odin slots (update) 
AP:       OneAI_AP_*.tar.md5
BL:       OneAI_BL_*.tar.md5
CP:       OneAI_CP_*.tar.md5
HOME_CSC: OneAI_HOME_CSC_*.tar.md5
3 Done!
  • Device reboots, apps and settings intact
  • If bootloop: TWRP → Wipe → Format Data → Reboot
